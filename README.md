## Setting UP

DATA NODE MANAGER requires Docker and Node to be installed

[https://docs.docker.com/engine/install/](https://docs.docker.com/engine/install/)

[https://nodejs.org/en/](https://nodejs.org/en/)


Install packages
### `npm install`


Run DATA NODE MANAGER
### `npm run build`
### `npm run manager`

Visit:
### http://127.0.0.1:8888/
